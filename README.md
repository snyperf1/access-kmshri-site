# Access Matters

Static website about web accessibility, built for deployment to `access.kmshri.com`.

## Files

- `index.html` - page structure and content
- `styles.css` - styles, layout, focus states, responsive behavior
- `script.js` - checklist progress + local storage persistence

## Local preview

Open `index.html` in a browser, or run a simple static server.

## Deployment

This project is intended for:

1. GitHub repository hosting
2. Vercel deployment
3. Custom domain mapping to `access.kmshri.com`
